# 隐私政策

**更新日期：2026-05-25**

## 概述

BeyondTranslation 是一款 Chrome 浏览器扩展程序，旨在帮助用户分析英语句子结构。本隐私政策说明我们如何处理用户数据。

## 数据收集与使用

### 1. 选中的文本

当您在网页上选中英文文本并右键点击「分析英文」时，选中的文本会被发送至我们的后端服务（通过阿里云函数计算调用 DeepSeek API），仅用于句法成分分析、翻译和例句生成。我们不会在服务器上存储这些文本。

### 2. 网页标注

「网页自动标注」功能会读取您当前访问的网页中的英文句子文本，发送至后端 AI 服务进行实时语法成分标注。我们不会收集或存储您的浏览历史或完整的网页内容。

### 3. 激活码

如果您购买了 Token 套餐，您会收到一个激活码。该激活码仅在激活时发送至我们的服务器进行有效性验证和 Token 余额管理。激活码不会用于追踪您的个人身份。

### 4. 设备标识

一个在浏览器本地随机生成的设备 ID 用于免费用户的每日使用额度追踪。该 ID 是匿名的，无法用于识别您的个人身份。

## 数据存储

- 所有设置、许可证状态和缓存数据均存储在浏览器本地的 `chrome.storage.local` 中。
- 我们不会将您的个人信息、浏览历史或任何可识别数据上传至我们的服务器。

## 第三方服务

AI 分析功能由 DeepSeek API 提供技术支持。您选中的文本将经过我们的后端转发至 DeepSeek 进行处理。请参阅 [DeepSeek 隐私政策](https://platform.deepseek.com/privacy) 了解其数据处理方式。

## 数据共享

我们不会向任何第三方出售、出租或共享您的个人数据。
