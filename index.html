安装步骤

1

下载扩展

下载 ZIP 包，解压到本地文件夹。

2

安装扩展

打开 Chrome，地址栏输入 chrome://extensions，开启「开发者模式」，点击「加载已解压的扩展程序」，选择解压后的文件夹。

3

配置 API Key

点击扩展图标 → 设置 → 填入 DeepSeek API Key。

4

输入激活码

在扩展侧边栏中输入购买后获得的激活码，充值 Token。
